# Custom WooCommerce Product Schema

Adds Product, Offer, AggregateRating, and Review schema markup to WooCommerce product pages without requiring premium SEO plugins.

## Description

Custom WooCommerce Product Schema generates Schema.org JSON-LD structured data for WooCommerce products to improve search engine understanding and eligibility for rich results.

The plugin outputs:

- Product Schema
- Brand Schema
- Offer Schema
- Aggregate Rating Schema
- Review Schema

It can use review data from BayReviews while also including predefined review entries.

## Features

- Product JSON-LD schema
- Offer schema with price and availability
- Brand schema
- Aggregate rating support
- BayReviews integration
- Static review support
- Automatic review count fallback
- Lightweight implementation
- No premium SEO plugin required

## Requirements

- WordPress 6.0+
- WooCommerce 6.0+
